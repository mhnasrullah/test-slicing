## Getting Started

Requierement node version:
node version >= 18.17.0

```bash
node -v
```

Install all package:

```bash
npm i
# or
yarn
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Account:

```bash
# ADMIN
admin@admin.com
admin123

# GUEST
guest@guest.com
guest123
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
